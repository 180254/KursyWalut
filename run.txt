IDE: Visual Studio 2015+(?)
It is UWP app, it requires Windows 10.

1. Add to your IDE new NuGet package source: Syncfusion.
NuGet configuration is described in Synfusion documentation.
https://help.syncfusion.com/uwp/installation-and-deployment#nuget-configuration

2. Choose proper build configuration (default is Debug ARM).
2.1 Rebuild solution.

3. It should work.
